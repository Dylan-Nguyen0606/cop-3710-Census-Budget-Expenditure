# cop-3710-Census-Budget-Expenditure

# My sources for this project are mainly limited to official US ".gov" websites wherein the data I am working with originates from those sites, particularly the https://census.gov site.
# The scope of this project is to utilize the census data released in order to create a public database that tracks how the government is spending money with the additional traits of tracking spending trends per category and add on indexed reporting views.
# The users of this project's output are likely to be data scientists or economists who need to document US spending habits to perform tasks such as predict economy outcomes or identify patterns that are occurring.
